# University Website

A responsive university main website built as an academic project using HTML, CSS, and JavaScript.

## Features
- Clean and organized layout
- Responsive design for different screen sizes
- Interactive navigation using JavaScript
- Modern CSS styling

## Technologies Used
- HTML
- CSS
- JavaScript

## About
This project was created as part of my studies at University of Applied Sciences, Ferizaj to practice frontend web development fundamentals.

## Installation
1. Clone the repository
2. Open `index.html` in your browser

No server setup needed - this is a static website.
